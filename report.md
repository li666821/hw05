## 2、核心知识梳理

### 1. 卷积神经网络（CNN）基础原理

CNN 是专为图像数据设计的深度学习模型，核心优势在于**利用局部感受野、权值共享、池化操作**大幅减少参数数量，同时高效提取图像特征，其核心组件包括：

| 卷积层（Convolutional Layer） |
| :----------------------- |
| 通过卷积核滑动提取图像局部特征（边缘、纹理等）  |

| 激活函数（ReLU）     |
| :------------- |
| 引入非线性，提升模型拟合能力 |

| 池化层（Max Pooling）          |
| :------------------------ |
| 下采样压缩特征图尺寸，减少计算量，同时保留关键特征 |

| 全连接层（Fully Connected Layer） |
| :-------------------------- |
| 将提取的特征映射到最终分类结果             |

### 2. 极简模型 SimpleCNN 结构

本文设计的 SimpleCNN 模型结构极简但高效，适配 MNIST 28×28 手写数字图像输入，结构如下：

1. **卷积层 1**：输入通道 1，输出通道 16，卷积核 3×3，步长 1，填充 1，输出特征图尺寸为`16×28×28`
2. **ReLU 激活**：引入非线性
3. **最大池化层**：池化核 2×2，步长 2，特征图尺寸压缩为`16×14×14`
4. **全连接层**：将`16×14×14`的特征展平为一维向量，映射到 10 个数字类别（0-9）

### 3. CNN核心结构要点与通用训练流程

#### （1）CNN核心结构要点

卷积神经网络是处理图像数据的专用深度神经网络，依靠多层结构堆叠实现分层特征提取，核心结构及核心特点如下：整体网络由卷积层、激活层、池化层、全连接层有序堆叠组成，区别于传统神经网络，具备局部感受野、权值共享、下采样三大核心特性，能够极大减少模型参数量，降低计算冗余。其中卷积层为核心特征提取模块，通过卷积核滑动扫描图像，逐层提取边缘、纹理、深层语义等图像特征；激活层通过非线性激活函数打破线性约束，赋予模型拟合复杂图像特征的能力；池化层通过下采样操作压缩特征图尺寸，精简参数、减少计算量，同时提升模型对图像平移、轻微形变的鲁棒性；全连接层负责整合全局特征，将二维特征图映射为一维特征向量，最终完成图像分类任务。本文的SimpleCNN模型是该通用结构的轻量化落地应用，适配手写数字简单图像识别场景。

#### （2）CNN通用训练流程

CNN模型的训练为迭代式闭环流程，适配所有图像分类任务，结合本次手写数字识别实践，通用训练流程可总结为标准化步骤：第一，数据集预处理与加载，完成数据集划分、数据归一化、数据增强及批量加载，统一数据维度与分布，提升训练稳定性；第二，模型前向传播，将批量图像数据输入网络，经卷积、激活、池化、全连接层逐层运算，输出类别预测结果；第三，损失计算，通过损失函数对比模型预测值与真实标签，量化模型预测误差；第四，反向传播与参数更新，基于梯度下降算法反向逐层求解参数梯度，通过优化器更新卷积核权重、偏置等网络参数，降低训练损失；第五，迭代训练与验证，设置多轮迭代训练，每轮训练后在验证集/测试集评估模型性能，配合正则化、学习率优化等方式抑制过拟合；第六，模型收敛与评估，持续迭代直至训练损失稳定、测试准确率不再提升，完成模型收敛，最终完成模型泛化能力综合评估。

### 4. 完整实现流程

#### （1）数据预处理

- 使用 MNIST 手写数字数据集，通过`torchvision`加载数据
- 数据标准化：使用 MNIST 数据集的均值`0.1307`和标准差`0.3081`进行归一化，提升模型训练稳定性
- 数据加载：使用`DataLoader`实现批处理，设置`batch_size=64`，并设置训练集与测试集划分

#### （2）模型训练与测试

- 训练流程：模型训练模式 → 前向传播计算损失 → 反向传播更新参数 → 循环多轮（epochs=5）
- 测试流程：模型评估模式 → 无梯度计算，仅前向传播验证模型泛化能力
- 损失函数：交叉熵损失（CrossEntropyLoss），适配多分类任务
- 优化器：Adam 优化器，学习率设置为`0.001`，自适应调整学习率，加速模型收敛

#### （3）结果可视化

- 数据集样本可视化：随机展示 MNIST 手写数字样本，直观理解数据分布
- 预测结果可视化：展示模型对测试集样本的预测结果，标注预测标签与真实标签
- 训练曲线可视化：绘制训练损失与测试准确率曲线，分析模型收敛过程

## 3、关键实践要点

1. **模型轻量化设计**：SimpleCNN 仅包含 1 层卷积 + 池化 + 1 层全连接，参数数量少，训练速度快，同时在 MNIST 上仍能达到极高准确率，体现了 CNN 在简单图像任务上的高效性。
2. **训练优化技巧**：
   1. 数据归一化：有效缓解梯度消失 / 爆炸问题，加快模型收敛
   2. 训练 / 测试模式切换：`model.train()`与`model.eval()`的正确使用，避免测试时模型参数被更新
   3. 无梯度测试：`with torch.no_grad()`减少内存占用，提升测试效率
3. **结果分析**：模型在 5 轮训练后即可达到 99%+ 的测试准确率，同时训练损失持续下降、测试准确率持续上升，说明模型无明显过拟合，泛化能力良好。

## 4、模型优化拓展方向

本文也提出了该极简模型的优化思路，为后续进阶学习提供方向：

1. **加深网络结构**：增加卷积层数量，引入更多池化层与全连接层，提升复杂特征提取能力
2. **正则化技术**：加入 Dropout 层、L2 正则化，缓解模型过拟合
3. **超参数调优**：调整卷积核数量、大小、步长，优化学习率、batch size 等超参数
4. **数据增强**：对 MNIST 数据进行旋转、平移、缩放等增强操作，提升模型鲁棒性

# 二. LeNet-5 网络结构说明

### 网络结构表

| 层类型    | 输入尺寸   | 输出尺寸  | 通道数 | 核大小 | 步长 | 填充 | 参数量       |
| ------ | ------ | ----- | --- | --- | -- | -- | --------- |
| 输入层    | 28x28  | 28x28 | 1   | -   | -  | -  | 0         |
| 卷积层1   | 28x28  | 24x24 | 6   | 5x5 | 1  | 0  | 156       |
| 池化层1   | 24x24  | 12x12 | 6   | 2x2 | 2  | 0  | 0         |
| 卷积层2   | 12x12  | 8x8   | 16  | 5x5 | 1  | 0  | 2416      |
| 池化层2   | 8x8    | 4x4   | 16  | 2x2 | 2  | 0  | 0         |
| 全连接层1  | 4x4x16 | 120   | -   | -   | -  | -  | 30840     |
| 全连接层2  | 120    | 84    | -   | -   | -  | -  | 10164     |
| 全连接层3  | 84     | 10    | -   | -   | -  | -  | 850       |
| **总计** | -      | -     | -   | -   | -  | -  | **44426** |

### 结构说明

LeNet-5网络由以下部分组成：

1. **卷积层1**：使用5x5卷积核，输出6个通道，用于提取基本特征
2. **池化层1**：使用2x2最大池化，步长为2，降低特征图尺寸
3. **卷积层2**：使用5x5卷积核，输出16个通道，提取更复杂的特征
4. **池化层2**：再次使用2x2最大池化，进一步降低特征图尺寸
5. **全连接层1**：将特征图展平后连接到120个神经元
6. **全连接层2**：连接到84个神经元
7. **全连接层3**：输出10个类别（对应数字0-9）

### 分析

- **准确率**：LeNet-5的测试准确率高于极简CNN，这是因为LeNet-5具有更深的网络结构和更多的参数，能够学习更复杂的特征表示。
